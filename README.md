
# 설치

- ```git clone https://github.com/choyeun/community-backend.git```
- ```git checkout develop/${your-name}```
- ```python -m venv .venv```
- ```pip install -r requrierments.txt```

# 실행

- ```uvicorn main:app --reload```

# 업로드

- ```git add .```
- ```commit -m "바뀐 점"```
- ```push origin develop/${your-name}```
